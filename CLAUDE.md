# CigarBrokerCRM

Standalone Windows desktop CRM for cigar brokerage, built with PySide6 + SQLAlchemy + SQLite. Dark themed (Whoastra style), branded Whoastra Labs LLC. Modules: Dashboard, Clients (CRUD + comm log + order history), Companies (CRUD + product catalog), Orders (line items, totals, status), Calendar, Reports (7 built-in reports with matplotlib charts, CSV export), Settings, Help. Data stored in `~/.cigarbrokercrm/cigarbroker.db`. Packaged as single-file exe via PyInstaller (`dist/CigarBrokerCRM.exe`). First launch starts empty ‚Äî no sample data.

## Project Structure
- `main.py` ‚Äî Entry point, main window with sidebar navigation + dashboard
- `app/database.py` ‚Äî SQLAlchemy models (Client, Company, Product, Order, OrderItem, Communication) + seed data
- `app/theme.py` ‚Äî Dark theme stylesheet
- `app/clients_tab.py` ‚Äî Clients CRUD + communication log + order history panel
- `app/companies_tab.py` ‚Äî Companies CRUD + product catalog management
- `app/orders_tab.py` ‚Äî Order entry with line items, status, totals
- `app/reports_tab.py` ‚Äî 6 reports with charts and CSV export
- `cigarbrokercrm.spec` ‚Äî PyInstaller build spec (single-file exe)
